# Overview
I am a Business Systems student at Rosebank College, part of Group 2. I am focused on understanding how information systems support business operations, decision-making, and organisational efficiency. I am building my knowledge in systems thinking, business processes, and the application of technology within business environments. My goal is to develop strong analytical and problem-solving skills that will allow me to contribute effectively to business and IT-driven solutions.

This project is a fashion website focused on timeless design shaped through simplicity, elegance, and intention. The brand was founded on the idea of creating clothing that is elegant, effortless, and enduring. Rather than following fast-changing trends, the website showcases pieces designed with clean silhouettes, high-quality materials, and thoughtful craftsmanship.

The collection balances minimalist aesthetics with contemporary influences, offering versatile designs that can be worn across different seasons, occasions, and personal styles. Every detail is carefully considered to create a space where simplicity and self-expression coexist.

The website was developed using HTML5, CSS3, and JavaScript to create a responsive and interactive user experience. Visual Studio Code was used as the development environment, while the Live Server extension was used for testing and previewing the website during development. Additional web technologies and techniques, including responsive design principles, SEO optimization, dynamic content functionality, animations, and interactive elements, were implemented to enhance usability and performance.
